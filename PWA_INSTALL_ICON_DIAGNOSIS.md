## ❌ 问题原因

**Chrome不显示安装图标的可能原因：**

1. **图标尺寸问题**
   - banner-1.jpg可能不是标准的PWA图标尺寸
   - Chrome要求至少一个192x192或512x512的图标
   - banner-1.jpg是横幅图片，不是正方形

2. **图标用途问题**
   - PWA图标应该是应用图标，不是横幅
   - Chrome可能检测到图标不合适

3. **Chrome缓存**
   - 浏览器可能缓存了旧的manifest
   - 需要清除缓存

4. **Chrome版本或设置**
   - 某些Chrome版本对PWA要求更严格
   - 可能需要满足更多条件

## 🔧 解决方案

### 方案1：创建真正的PWA图标（推荐）

**步骤：**

1. **创建一个简单的PNG图标**
   - 512x512像素
   - 正方形
   - 简单的"L"字母或Luna logo

2. **更新manifest.json**
   ```json
   "icons": [
     {
       "src": "/icon-192.png",
       "sizes": "192x192",
       "type": "image/png",
       "purpose": "any maskable"
     },
     {
       "src": "/icon-512.png",
       "sizes": "512x512",
       "type": "image/png",
       "purpose": "any maskable"
     }
   ]
   ```

3. **重新部署**

### 方案2：使用Chrome菜单安装（当前可用）

**即使地址栏没有图标，你仍然可以安装PWA：**

**步骤：**
1. 点击Chrome右上角的三个点菜单（⋮）
2. 查找"安装LUNA"或"Install LUNA"选项
3. 点击安装

**这个方法100%可用！**

### 方案3：使用页面内的安装按钮（当前可用）

**页面右下角有安装提示：**
1. 点击"Try Auto-Install"按钮
2. 在浏览器弹出的对话框中确认
3. PWA会安装

### 方案4：清除Chrome缓存后重试

**步骤：**
1. 按Ctrl+Shift+Delete
2. 选择"缓存的图片和文件"
3. 清除
4. 刷新页面（Ctrl+Shift+R）
5. 等待10秒
6. 查看地址栏
