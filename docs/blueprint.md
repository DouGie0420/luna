# **App Name**: Luna Exchange

## Core Features:

- User Authentication & KYC: Secure user registration, login, and KYC verification.
- Product Listing & Purchasing: Allow users to list and purchase items with detailed descriptions and images.
- Admin Dashboard with Role-Based Access: Backend management system with different permission levels for administrators, employees, and customer support. Includes functionalities to manage users, products, orders and support interactions
- Multi-Currency Escrow System: Facilitate secure transactions using escrow accounts with support for THB, USDT, Alipay(RMB), WechatPay(RMB) and direct transfers. Later integrations with Claude can assist in automation
- Integrated Chat System: Embed Line or Telegram for user-to-user communication and provide a ticket system for user-to-customer support interactions.
- Location-Based Recommendations: Implement a location service to display products within a certain radius and use AI to provide product recommendations tailored to user interest based on purchase history and geographical proximity.
- Smart Search: Suggest and rank results as users type, while learning user search history.

## Style Guidelines:

- Primary color: Soft, lunar yellow (#FAF8E7) evoking a sense of trustworthiness and connection to the app's name, Luna.
- Background color: Very light yellow (#FCFDF7), maintaining consistency with the primary, but creating a subtle, calm background.
- Accent color: Muted beige (#E3D9C2) to highlight key actions and interactive elements, in a tone related to both the yellow tones chosen for primary/background.
- Body font: 'Inter', sans-serif, known for its readability and neutral appearance.
- Headline font: 'Space Grotesk', sans-serif, a modern typeface to bring tech vibes. Use 'Space Grotesk' for headlines and short text, combined with 'Inter' for longer text.
- Minimalist, line-based icons with a subtle, modern aesthetic. Use icons with rounded corners to match the overall gentle feeling.
- Use smooth, subtle transitions and animations to enhance user experience. Employ minimalist loading animations that are visually unobtrusive.
