## 🚀 使用指南

### 访问管理后台

1. 确保你的账号是管理员
   - 在Firestore中设置 `users/{userId}/role = 'admin'`
   - 或设置 `users/{userId}/isAdmin = true`

2. 访问 `/admin`
   - 系统会自动检查权限
   - 非管理员会被拒绝访问

3. 查看统计数据
   - 首页显示所有统计数据
   - 数据实时从Firestore读取

4. 配置支付方式
   - 访问 `/admin/settings`
   - 开启/关闭支付方式
   - 点击"Save Settings"保存
