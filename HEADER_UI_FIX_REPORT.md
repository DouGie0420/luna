## 🎯 设计原则

### 1. 统一性
- 所有按钮使用相同的结构
- 统一的圆角（rounded-full）
- 统一的内边距（px-4 py-2）
- 统一的发光效果

### 2. 层次感
- 主要操作：彩色渐变 + 发光
- 次要操作：白色半透明
- 管理功能：特殊颜色（黄色）

### 3. 可识别性
- 每个按钮有独特的颜色
- 每个按钮有对应的图标
- 重要按钮有发光效果

### 4. 交互反馈
- hover时边框变亮
- hover时发光效果出现
- 点击后有视觉反馈
