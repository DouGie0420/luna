# 🚀 15美元加密货币高频交易系统

一个专为小额资金设计的自动化加密货币交易系统，重点关注比特币5分钟级别的短线机会。

## ✨ 核心功能

- **Twitter信号监控**：追踪关键影响者和交易员的实时讨论
- **技术分析**：RSI、布林带、移动平均线等指标
- **价格突增检测**：5分钟内涨跌幅超过3%的快速识别
- **自动风险管理**：凯利公式仓位管理，严格止损止盈
- **高频交易优化**：冷却时间控制，防止过度交易

## ⚠️ 重要警告

**15美元本金交易加密货币极度高风险**：
- Gas费可能吃掉大部分利润
- 市场波动可能导致快速亏损
- 高频交易需要极低延迟
- 仅建议使用可承受完全损失的资金

## 🛠️ 快速开始

### 1. 环境准备
`ash
cd arbitrage-tools
npm install
`

### 2. 配置API密钥
复制环境变量模板：
`ash
cp .env.example .env
`

编辑.env文件，填入以下API密钥：

#### 必需配置：
- **Twitter开发者账号**：[申请地址](https://developer.twitter.com/)
- **币安API**（或其他交易所）

#### 可选配置：
- **Telegram Bot**：用于接收交易警报
- **Alchemy/Infura**：链上数据（如果需要）

### 3. 调整交易参数
在.env中修改：
`env
INITIAL_CAPITAL=15           # 初始资金（美元）
TRADING_PAIR=BTC/USDT        # 交易对
STOP_LOSS=0.02               # 2%止损
TAKE_PROFIT=0.05             # 5%止盈
MAX_DAILY_TRADES=10          # 每日最大交易次数
`

### 4. 启动系统
`ash
npm start
`

### 5. 监控交易
系统每5分钟输出状态报告，包括：
- 当前余额和盈亏
- 胜率和交易次数
- 活跃信号和交易

## 🎯 交易策略

### 信号来源
1. **Twitter情绪分析**：追踪Elon Musk、CZ等关键人物
2. **技术指标**：RSI超买超卖，布林带突破
3. **价格突增**：5分钟内涨跌幅>3%

### 风险管理
- **凯利公式仓位管理**：根据胜率动态调整仓位
- **硬性止损**：单笔交易最大亏损2%
- **每日限额**：最多10笔交易，最大回撤50%
- **冷却时间**：交易后5分钟内不接收新信号

### 执行逻辑
`
信号产生 → 置信度检查 → 仓位计算 → 开仓
              ↓
        价格监控循环
              ↓
    止损/止盈检查 → 平仓
`

## 🚨 风险控制

### 自动停止条件
1. **余额低于**：自动停止所有交易
2. **单日亏损+**：暂停当日交易
3. **连续3次亏损**：进入冷却模式
4. **网络异常**：暂停交易直到恢复

### 手动干预
- 随时可以发送SIGINT(Ctrl+C)停止系统
- 交易记录保存在data/trade-state.json
- 可以修改配置后重新启动

## 📈 性能优化建议

1. **使用VPS**：降低延迟，24/7运行
2. **多交易所**：分散风险，获取更好价格
3. **Gas优化**：在Base链Gas低时交易
4. **回测验证**：先用历史数据测试策略
5. **渐进投入**：先模拟交易，再小资金测试
